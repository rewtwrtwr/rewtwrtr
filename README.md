## Что здесь

Репозитории с учебными проектами:

- **Скрипты для журналов ED** — учусь парсить JSON-логи игры, извлекать данные о полетах и торговле
- **Тесты API** — пробую автоматизировать проверки для EDSM/Inara, учусь работать с HTTP и JSON
- **EDMC-плагины** — экспериментирую с Elite Dangerous Market Connector, разбираюсь в их SDK
- **Утилиты автоматизации** — тестирую библиотеки для работы с окнами и вводом

Всё в процессе обучения — код может быть неидеальным или незаконченным.
